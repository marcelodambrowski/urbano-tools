1
Extração do pacote
Executar urbano-tools.exe

O arquivo urbano-tools.exe é um pacote autoextraível (self-extracting archive). Ele não é o programa em si — é um instalador compactado que contém todos os arquivos LISP, DLLs e recursos do Urbano Tools.

  • 1Dê duplo clique em urbano-tools.exe.
  • 2Quando solicitado o destino da extração, informe (ou confirme) exatamente o caminho abaixo.
C:\urbano-tools
  • 3Confirme e aguarde a extração terminar.
O caminho C:\urbano-tools é fixo e obrigatório. Os arquivos .bat de configuração e as rotinas LISP fazem referência direta a esse caminho — se a extração for feita em outra pasta, a configuração automática do AutoCAD e o carregamento das rotinas irão falhar.
2
Configuração do AutoCAD
Informar o Support File Search Path

É necessário informar ao AutoCAD onde estão os arquivos de suporte do Urbano Tools. Existem dois caminhos, dependendo do produto instalado:

2.1 — Se você utiliza AutoCAD Civil 3D (2023 a 2027)

Dentro da pasta C:\urbano-tools está o arquivo _1.configurar_Civil3D.bat, que automatiza a configuração.

  • 1Execute o arquivo _1.configurar_Civil3D.bat (não é necessário abrir como administrador).
  • 2O script varre o registro do Windows e detecta automaticamente quais versões do Civil 3D (2023, 2024, 2025, 2026 ou 2027) estão instaladas e qual já está configurada.
  • 3Ele exibe um resumo das versões encontradas e pergunta se deseja aplicar a configuração (S/N).
  • 4Ao confirmar com S, o script adiciona C:\urbano-tools ao Support File Search Path de cada versão detectada, direto no perfil <<C3D_Metric>> do AutoCAD.
  • 5Reinicie o Civil 3D — as ferramentas Urbano Tools estarão disponíveis automaticamente.
Para conferir o resultado, vá em Civil 3D: Options > Files > Support File Search Path e confirme se C:\urbano-tools aparece na lista.
2.2 — Se você utiliza AutoCAD "puro" (sem Civil 3D)

O script _1.configurar_Civil3D.bat só reconhece perfis do Civil 3D. Em um AutoCAD comum, adicione o caminho manualmente:

  • 1Abra o AutoCAD normalmente.
  • 2Digite o comando OPTIONS (ou vá em Ferramentas/Tools > Options) e pressione Enter.
  • 3Clique na aba Files (Arquivos).
  • 4Localize e expanda Support File Search Path.
  • 5Clique com o botão direito e escolha Add (Adicionar), ou use o botão Add na parte inferior.
  • 6Cole ou digite exatamente o caminho abaixo.
C:\urbano-tools
  • 7Clique em Apply (Aplicar) e depois em OK.
  • 8Feche e reabra o AutoCAD para que o caminho seja reconhecido.
Esse mesmo procedimento manual também pode ser usado em qualquer versão do Civil 3D, caso o script automático não funcione (permissão do Windows, perfil de desenho diferente do padrão, etc.).
3
Instalação do Python
Baixar e instalar o Python (versão mais recente)

Diversas ferramentas do Urbano Tools (otimizador de GeoTIFF, extrator de memoriais SIGEF) dependem de scripts Python executados em segundo plano pelas rotinas LISP. Por isso, o Python precisa estar instalado e acessível pelo sistema.

O Python não vem incluído no pacote (é um arquivo grande). Baixe direto do site oficial pelo botão abaixo.
Baixar Python (site oficial)
  • 1Clique no botão acima — ele abre a página oficial de downloads do Python para Windows.
  • 2Baixe sempre a versão estável mais recente disponível (64-bit), no topo da página — não é necessário procurar uma versão específica.
  • 3Execute o instalador baixado (ex: python-3.x.x-amd64.exe).
  • 4Na primeira tela, marque obrigatoriamente a caixa "Add python.exe to PATH" (ou "Add Python to PATH"), na parte inferior da janela, antes de clicar em Install Now.
  • 5Clique em Install Now e aguarde a conclusão.
  • 6Se aparecer "Disable path length limit" ao final, recomenda-se clicar também para desabilitar o limite de caminho do Windows.
Marcar "Add to PATH" é essencial. Sem essa opção, o Windows não encontra o comando python a partir do AutoCAD/LISP, e todas as ferramentas que dependem de Python vão falhar silenciosamente ou apresentar erro de execução.

Caso o Python já tenha sido instalado sem marcar essa opção: execute novamente o instalador, escolha Modify e, na etapa seguinte, marque Add Python to environment variables antes de concluir.

4
Instalação das bibliotecas Python
Executar _install_bibliotecas_python.bat

Com o Python já instalado e no PATH (Etapa 3), o próximo passo é instalar as bibliotecas Python que dão suporte às ferramentas do Urbano Tools.

  • 1Execute o arquivo _install_bibliotecas_python.bat.
  • 2O script atualiza o pip (gerenciador de pacotes do Python) para a versão mais recente.
  • 3Em seguida instala, em sequência, as bibliotecas abaixo.
pyproj — coordenadas SIRGAS2000/UTM pdfplumber — leitura de PDFs SIGEF rasterio — otimização de GeoTIFF
  • 4Ao final, a janela exibirá "Instalacao concluida com sucesso!" — pressione qualquer tecla para fechar.
Se o script apresentar erro de comando python não reconhecido, é sinal de que a Etapa 3 (Add to PATH) não foi concluída corretamente. Reinstale o Python marcando essa opção e execute este .bat novamente.
5
Ativação e renovação da licença
Sistema Porteira — automático

O Urbano Tools usa um sistema de licenciamento próprio (chamado internamente de Porteira), carregado automaticamente toda vez que um desenho é aberto no AutoCAD, após a configuração da Etapa 2. Não é preciso digitar nenhum comando.

  • 1Ao abrir qualquer desenho, a Porteira identifica o computador através de um número de série único do disco rígido (ID de Cliente).
  • 2Esse ID é conferido automaticamente contra a lista de licenças válidas pela internet.
  • 3Se a licença estiver válida e dentro do prazo, o Urbano Tools carrega normalmente e a Ribbon com as ferramentas aparece no AutoCAD.
  • 4Se a licença ainda não tiver sido ativada, ou já tiver expirado, uma janela de aviso é exibida, mostrando o ID de Cliente e perguntando se deseja abrir o suporte via WhatsApp.
  • 5Ao confirmar, o AutoCAD abre automaticamente uma conversa de WhatsApp com o suporte, já com a mensagem pronta contendo o ID de Cliente.
  • 6Após a confirmação do pagamento/renovação, a liberação é feita remotamente — basta fechar e abrir o AutoCAD novamente, sem reinstalar nada.
Guarde o ID de Cliente exibido na janela de ativação: ele identifica o computador para fins de licenciamento e será solicitado sempre que for necessário ativar, renovar ou trocar o Urbano Tools de máquina.
Funciona também sem internet. Mesmo offline, a Porteira consegue verificar a versão da licença já validada anteriormente no computador e libera o carregamento normal do Urbano Tools. A conferência online volta a acontecer automaticamente assim que a internet estiver disponível novamente.

Dúvidas frequentes
Perguntas comuns na instalação
Posso extrair em outra pasta, diferente de C:\urbano-tools?
Não é recomendado. Os scripts de configuração e as rotinas LISP referenciam esse caminho diretamente. Extrair em outro local exige ajustar manualmente cada referência e pode quebrar o auto-update.
O script _1.configurar_Civil3D.bat não encontrou nenhuma versão. E agora?
Isso indica que você não tem o Civil 3D instalado (só AutoCAD comum) ou que o Civil 3D está em um perfil não-padrão. Siga o procedimento manual da Etapa 2.2 para adicionar o caminho pela janela Options do AutoCAD.
Já instalei o Python antes, preciso reinstalar?
Só se a opção "Add to PATH" não tiver sido marcada na instalação original. Nesse caso, rode o instalador novamente, escolha Modify e marque "Add Python to environment variables".
Preciso ativar a licença manualmente todo dia?
Não. Uma vez ativada, a verificação acontece automaticamente toda vez que o AutoCAD abre um desenho, sem nenhuma ação do usuário — só é exibido aviso se a licença estiver expirada ou ainda não ativada.
