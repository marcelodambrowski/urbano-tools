1
Extração do pacote
Extrair urbano-tools-main.zip

O arquivo urbano-tools-main.zip contém todos os arquivos necessários e recursos do Urbano Tools.

  • 1Clique com o botão direito em urbano-tools-main.zip e escolha Extrair tudo...
  • 2Quando solicitado o destino da extração, informe ou confirme exatamente o caminho abaixo.
C:\urbano-tools
  • 3Clique em Extrair e aguarde a extração terminar.
O caminho C:\urbano-tools é fixo e obrigatório. Os arquivos e as rotinas LISP fazem referência direta a esse caminho — se a extração for feita em outra pasta, o carregamento do urbano tools e o seu funcionamento irá falhar.
2
Configuração do AutoCAD
Informar o Support File Search Path

O AutoCAD precisa saber onde estão os arquivos do Urbano Tools. Escolha a opção conforme o que você usa:

Versão Civil 3D — automático
  • 1Dentro de C:\urbano-tools, execute o arquivo _1.configurar_Civil3D.bat.
  • 2O script detecta as versões instaladas e pergunta se deseja aplicar a configuração (S/N).
  • 3Confirme com S e pronto, já está configurado.
Versão AutoCAD "puro" (ou se o script não funcionar)

Adicione o caminho manualmente pela janela Options:

  • 1Digite OPTIONS no AutoCAD e abra a aba Files.
  • 2Expanda Support File Search Path, clique em Add e cole o caminho abaixo.
C:\urbano-tools
  • 3Clique em Apply, depois em OK.
Para conferir se está tudo certo, vá novamente em Options > Files > Support File Search Path e confirme se C:\urbano-tools aparece na lista.
3
Instalação do Python
Baixar e instalar o Python (versão mais recente)

Diversas ferramentas do Urbano Tools dependem de scripts Python executados pelas rotinas LISP. Por isso, o Python precisa estar instalado e acessível pelo sistema.

O Python não vem incluído no pacote. Baixe direto do site oficial pelo botão abaixo.
Baixar Python (site oficial)
  • 1Clique no botão acima e baixe a versão estável mais recente disponível para Windows — não é necessário procurar uma versão específica.
  • 2Execute o instalador baixado.
  • 3Na primeira tela, marque obrigatoriamente a caixa "Add python.exe to PATH" antes de clicar em Install Now.
  • 4Clique em Install Now e aguarde a conclusão.
Marcar "Add to PATH" é essencial. Sem essa opção, o Windows não encontra o comando python a partir do AutoCAD/LISP, e as ferramentas que dependem de Python vão falhar.

Se o Python já estiver instalado sem essa opção marcada: rode o instalador de novo, escolha Modify e marque Add Python to environment variables.

4
Instalação das bibliotecas Python
Executar _2.install_bibliotecas_python.bat

Com o Python já instalado e no PATH (Etapa 3), o próximo passo é instalar as bibliotecas Python que dão suporte às ferramentas do Urbano Tools.

  • 1Dentro de C:\urbano-tools, execute o arquivo _2.installLbibliotecas_python.bat.
  • 2O script atualiza o pip (gerenciador de pacotes do Python) para a versão mais recente.
  • 3Em na sequência instala as bibliotecas abaixo.
pyproj — coordenadas SIRGAS2000/UTM pdfplumber — leitura de PDFs SIGEF rasterio — otimização de GeoTIFF
  • 4Ao final, a janela exibirá "Instalacao concluida com sucesso!" — pressione qualquer tecla para fechar.
Se o script apresentar erro de comando python não reconhecido, é sinal de que a Etapa 3 (Add to PATH) não foi concluída corretamente. Reinstale o Python marcando essa opção e execute este .bat novamente.
5
Ativação e renovação da licença
Sistema Porteira — automático

O Urbano Tools usa um sistema de licenciamento próprio chamado internamente de Porteira, carregado automaticamente ao abrir um desenho no AutoCAD. Não é preciso digitar nenhum comando.

  • 1Ao abrir um desenho CAD, a Porteira identifica o computador e confere sua licença automaticamente no servidor.
  • 2Se estiver válida, o Urbano Tools carrega normalmente e a Ribbon aparece no AutoCAD.
  • 3Se ainda não estiver ativada ou tiver expirada, uma janela será aberta com o seu numero de ID e dará opção de abrir o suporte via WhatsApp. Para ativar/reativar a licença.
  • 4Após a renovação, a liberação é feita remotamente — basta apenas reabrir o AutoCAD.
Funciona também sem internet. A Porteira usa a última verificação válida no computador e volta a conferir online assim que a internet estiver disponível.
Falar com o suporte para ativar
Desinstalação: para remover o Urbano Tools, apague a pasta C:\urbano-tools e, se o caminho foi adicionado manualmente, retire-o em OPTIONS > Files > Support File Search Path.
Dúvidas frequentes
Perguntas comuns na instalação
Posso extrair em outra pasta, diferente de C:\urbano-tools?
Não! Os scripts de configuração e as rotinas LISP referenciam esse caminho diretamente. Extrair em outro local quebrar todo o sistema.
O script _1.configurar_Civil3D.bat não encontrou nenhuma versão. E agora?
Isso indica que você não tem o Civil 3D instalado, apenas o AutoCAD comum ou que o Civil 3D está em um perfil não-padrão. Siga o procedimento manual da Etapa 2 para adicionar o caminho pela janela Options do AutoCAD.
Já instalei o Python antes, preciso reinstalar?
Só se a opção "Add to PATH" não tiver sido marcada na instalação original. Nesse caso, rode o instalador novamente, escolha Modify e marque "Add Python to environment variables".
Preciso ativar a licença manualmente todo dia?
Não. Uma vez ativada, a verificação acontece automaticamente toda vez que o AutoCAD abre um desenho, sem nenhuma ação do usuário — só é exibido aviso se a licença estiver expirada ou ainda não ativada.
